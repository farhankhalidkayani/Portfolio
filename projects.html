DiceRoller CLI

DiceRoller CLI is a command-line application for rolling dice with customizable parameters. Users can specify the number of dice and the number of faces per die using a simple format (e.g., 3d6 for three six-sided dice). The program supports rolling the dice multiple times and logging the results to an external file. DiceRoller CLI is ideal for gamers, statisticians, and anyone who needs a reliable dice rolling tool.

Spotify Clone is a web-based music player designed to replicate the core features of Spotify’s user interface using HTML and CSS.
